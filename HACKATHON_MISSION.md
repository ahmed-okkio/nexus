# Hackathon Mission: Nexus AI Assistant

This project is being built for the **LOS Hackathon (May 9th - 16th, 2026)**.

## Project Vision
Nexus is a personal AI assistant that acts as an extension of the user's mind, capturing, organizing, and acting on notes and tasks via a natural conversational interface.

## Core Requirements
- **Must Have:**
  - Conversational Note Taking (thoughts, ideas, meetings).
  - Task Management (create, update, track, complete).
  - Natural, conversational UX (feels like talking to an assistant).
- **Should Have:**
  - Reminders.
  - Intelligent Retrieval (querying history).
  - Daily Briefing (command-line summary of tasks/notes).
- **Stretch Goals:**
  - Calendar Integration.
  - Proactive Suggestions.
  - Cross-linking (notes ↔ tasks ↔ calendar).

## Development Constraints
- **CLI-based Development:** All agentic interaction must occur via the command line.
- **Free Tier:** Limited to free-tier API access (Gemini, Copilot, etc.).

## Evaluation Criteria
Our performance will be judged by an AI panel based on:
1. Product completeness (25%).
2. Demo quality (25%).
3. **Agentic Sophistication (25%):** Use of advanced paradigms (planning, sub-agents, multi-step execution, token efficiency).
4. Token efficiency (15%).
5. Creativity & Innovation (10%).

## Key Paradigms
We are prioritizing **Planning**, **Agent Skills**, **Persistent Memory**, and **Token Efficiency** throughout our development lifecycle.
