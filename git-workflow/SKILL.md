# Git Workflow Skill

Use this skill to perform daily git operations efficiently.

## Routine Operations

- **Branching**: `git checkout -b <branch-name>`
- **Committing**: `git add .` followed by `git commit -m "<message>"`
- **Pushing**: `git push -u origin <branch-name>`

## Usage

When you have changes to commit, say "commit these changes with message 'update notification hook'". To start a new task, say "create a feature branch named 'add-new-feature'".
